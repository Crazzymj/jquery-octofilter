# jQuery Octo Filter

A jQuery autocomplete categorized plugin

## Getting Started

In your web page:

```html
<script src="jquery.js"></script>
<script src="dist/jquery-octo-filter.min.js"></script>
<script>
  jQuery(function($) {
    $('input').octofilter({
      url: 'data.jon'
    });
  });
</script>
```
